# 프로젝트 개요

## 군대 당직 관리 프로그램
### 배경
본 프로젝트 작성자는 군 복무 시 항상 수작업으로 당직을 작성하는 것이 불필요한 행정업무라고 생각했음
당직순서가 따로 정해져 있었기 때문에
관리자는 원하는 기간을 지정하면 자동으로 당직표가 나오는 프로그램이 있으면
행정업무 부담을 줄일 수 있다고 생각됨.
---
## 사용 라이브러리
* Java21
* Apache POI
* gradel
* 실행환경 : Window10, 11, Mac OS
---
## 주요 기능

### 튜토리얼 기능
1. [x] 당직 순서 양식 파일을 생성해 다운 받는다.
   * [x] 기본 정보가 들어가 있는 파일
2. [x] 생성된 파일을 읽어서 최종 결과물을 생성할 수 있다.
3. 기능 정보에 관한 상세한 문서를 엑셀로 다운 받을 수 있다.

### 입력 기능
* [x] 엑셀 파일을 입력 받을 수 있어야 한다.
* [x] 원하는 당직표 기간을 입력 받을 수 있다.

### 출력 관련 기능
* [x] 튜토리얼 관련 기능을 충족할 수 있어야 한다.
* [x] 최종 결과물을 하나의 엑셀 파일로 새로 생성해 지정된 폴더에 저장할 수 있어야 한다.
  * [x] 1번 Sheet : 최종 당직표
  * [x] 2번 Sheet : 당직자 순서(평일)
  * [x] 3번 Sheet : 당직자 순서(휴일)
* [x] 최종 결과물 파일(엑셀)을 프로그램이 있는 디렉터리 하위에 새로운 폴더를 만들어서 저장한다.
  * [x] 예) 당직관리 폴더/2025년/01월/당직표_2025년_01월_22일_16시_30분.xlsx
  * [x] 최종 결과 파일은 항상 새로운 파일을 생성해서 저장해야 한다.

### 당직 순서 관련 기능
* [x] 평일 당직은 평일 당직 순서에 따른다.
* [x] 휴일 당직은 휴일 당직 순서에 따른다.

### 법정휴무일 기능
* [x] 엑셀 파일로 각 년도 법정 휴무일 데이터를 미리 가지고 있는다.
* [x] 관리자는 holidays.xlsx 내부 휴무일 데이터를 추가 할 수 있다.
* [x] 관리자는 holidays.xlsx 내부 휴무일 데이터를 삭제할 수 있다.
* [x] 휴무일 데이터는 오직 holidays.xlsx을 통해서 관리한다.

### 평일, 휴일 변경 기능
 * [x] 관리자는 평일을 휴일로 변경할 수 있다.
   * [x] 휴일로 변경시 휴무일 엑셀 DB에 내용이 반영되어야 한다.
 * [x] 관리자는 휴일을 평일로 변경할 수 있다.

### 당직 변경 기능
* [x] 당직 상호 변경 기능
* [x] 당직 단일 변경 기능
* [x] 당직 변경은 평일은 평일끼리, 휴일은 휴일끼리 하도록 검증 기능

### 연속 당직 금지 기능
* [x] 연속 당직은 강제적으로 앞,뒤 순서를 변경시킨다.
  * [x] 평일은 평일끼리
    1. 변경 전 : 목: 김해군, 금: 김해군, 월: 최왕건
    2. 변경 후 : 목: 김해군, 금: **최왕건**, 월: **김해군**
  * [x] 휴일은 휴일끼리
    1. 변경 전 : 금(공휴일): 김해군, 토: 김해군, 일: 최왕건
    2. 변경 후 : 금(공휴일): 김해군, 토: **최왕건**, 일: **김해군**
