# 🎯프리코스 오픈 미션 과제 - 군대 당직 관리 프로그램

## 🪖군대 당직 관리 프로그램 소개🪖
### 배경
해당 프로그램은 군대에서 수작업으로 매주, 매월 당직표를 작성하는 것이 불필요하고 반복적인 행정업무라고 생각했다.
평일/휴일 각각 당직자 순서가 정해져 있었기 때문에, 당직 관리자는 원하는 기간을 지정하면 자동으로 당직표가 나오는 프로그램이 있으면
행정업무 부담을 줄일 수 있다고 평소에 생각해왔다. 그래서 프리코스 오픈미션에 평소에 생각만 해왔던 일상의 도움을 주는 프로그램을 만들기로 정했습니다.

## 주요 기능

### 튜토리얼 기능
1. [x] 당직자 순서(양식) 파일을 생성해 다운 받는다.
   * [x] 기본 정보가 들어가 있는 파일
2. [x] 공휴일 데이터베이스를 생성할 수 있다
   * [x] 기존 DB가 존재할 시 덮어씌우기 여부를 물어본다.
   * [x] 기존 DB를 덮어씌울시 이전 DB는 백업파일로 남겨둔다.
3. [x] 생성된 파일을 읽어서 최종 결과물을 생성할 수 있다.

### 입력 기능
* [x] 엑셀 파일을 입력 받을 수 있어야 한다.
* [x] 원하는 당직표 기간을 입력 받을 수 있다.
* [x] [error] 잘못된 년,월,일 입력에 에러 메시지를 화면에 전시할 수 있다.
* [x] [error] 잘못된 파일을 입력받을 시 에러 메시지를 화면에 전시할 수 있다.

### 출력 관련 기능
* [x] 튜토리얼 관련 기능을 충족할 수 있어야 한다.
* [x] 최종 결과물을 하나의 엑셀 파일로 새로 생성해 지정된 폴더에 저장할 수 있어야 한다.
  * [x] 1번 Sheet : 최종 당직표
  * [x] 2번 Sheet : 당직자 순서(평일)
  * [x] 3번 Sheet : 당직자 순서(휴일)
* [x] 최종 결과물 파일(엑셀)을 바탕화면에 "당직 관리 프로그램" 폴더를 만들어서 저장한다.
  * [x] 예) 당직 관리 프로그램/2025년/01월/당직표_2025년_01월_22일_16시_30분.xlsx

### 당직 순서 관련 기능
* [x] 평일 당직은 평일 당직 순서에 따른다.
* [x] 휴일 당직은 휴일 당직 순서에 따른다.
* [x] 공휴일 DB를 자동으로 감지해 휴일 당직 순서를 적용한다.

### 공휴일 데이터베이스 관리 기능
* [x] 기본 공휴일 DB를 생성할 수 있다.
  * [x] DB 생성시 자동으로 파일이 열린다.
  * [x] 기본 DB에는 기본 데이터가 들어 있다.(2025년, 2026년 공휴일)
  * [x] DB 생성시 기존에 DB 파일이 있는지 확인할 수 있다.
  * [x] DB 생성시 기존 DB 파일이 있다면 경고 메시지를 전시할 수 있다.
  * [x] DB 생성시 이전 DB는 백업파일로 저장한다. 
* [x] 관리자는 holidays.xlsx 내부 휴무일 데이터를 추가 할 수 있다.
  * * [x] [ERROR] 잘못된 양식의 데이터 추가시 에러 메시지를 전시한다.
* [x] 관리자는 holidays.xlsx 내부 휴무일 데이터를 삭제할 수 있다.
* * [x] [ERROR] 잘못된 양식의 데이터 삭제시 에러 메시지를 전시한다.
* [x] 휴무일 데이터는 오직 holidays.xlsx을 통해서 관리한다.
* [x] [ERROR] 공휴일 DB 파일이 없으면 에러 메시지를 전시한다.

### 평일, 휴일 변경 기능
 * [x] 관리자는 평일을 휴일로 변경할 수 있다.
   * [x] 휴일로 변경시 휴무일 엑셀 DB에 내용이 반영되어야 한다.
 * [x] 관리자는 휴일을 평일로 변경할 수 있다.

### 연속 당직 금지 기능
* [x] 연속 당직은 강제적으로 앞,뒤 순서를 변경시킨다.
  * [x] 평일은 평일끼리
    1. 변경 전 : 목: 김해군, 금: 김해군, 월: 최왕건
    2. 변경 후 : 목: 김해군, 금: **최왕건**, 월: **김해군**
  * [x] 휴일은 휴일끼리
    1. 변경 전 : 금(공휴일): 김해군, 토: 김해군, 일: 최왕건
    2. 변경 후 : 금(공휴일): 김해군, 토: **최왕건**, 일: **김해군**
